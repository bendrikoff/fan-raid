# Mini-game backgrounds

Place custom JPG backgrounds here with these exact names:

- `feature.jpg` - large top game block
- `penalty.jpg`
- `event-predictor.jpg`
- `keeper-reaction.jpg`
- `corner-kick.jpg`
- `football-quiz.jpg`
- `quick-penalty.jpg`
- `quick-event-predictor.jpg`
- `quick-keeper-reaction.jpg`
- `quick-corner-kick.jpg`
- `quick-football-quiz.jpg`

Recommended size: 1200x700 for `feature.jpg`, 600x420 for game cards, 340x220 for quick-start thumbnails.
If a file is missing, the UI visually falls back to `../live-match-bg.jpg`.
